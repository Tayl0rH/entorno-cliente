• URL Dirección única que identifica y localiza un recurso(archivo html o php, imagenes...) específico en Internet.
  • HTTP Protocolo de comunicación que usan los navegadores y servidores para inercambiar información.
  • TCP/IP Conjunto de protocolos que define cómo se divide, envía, dirige y recibe la información entre dispositivos y redes, incluyendo Internet. TCP se encarga de garantizar que los datos lleguen correctamente, en orden, y sin pérdidas; IP de direcciones y enrutar los paquetes hasta su destino.
  • Internet Sistema global de redes de computadoras interconectadas que utilizan TCP/IP para comunicarse.
  • W3C Organización internacional sin fines de lucro que desarrolla estándares abiertos para asegurar que la Web sea accesible, interoperable y evolucione de forma organizada.
  • HTML Lenguaje de marcado estándar para crear y estructurar páginas web.
